Step 3: Apply Clustering Algorithms

K-Means Clustering

Partition-based algorithm that minimizes within-cluster variance.

K-Medoids Clustering

Like K-Means but uses actual data points (medoids) as cluster centers.

AGNES (Agglomerative Hierarchical Clustering)

Bottom-up hierarchical algorithm starting from individual points and merging them.

DIANA (Divisive Hierarchical Clustering)

Top-down hierarchical algorithm starting from all points together and splitting them.

K-Distance Graph

Generate a K-distance graph to find optimal epsilon for DBSCAN.

DBSCAN (Density-Based Clustering)

Finds clusters of arbitrary shape based on data density.

Comparison

Compare the performance of all applied algorithms using standard evaluation metrics.
